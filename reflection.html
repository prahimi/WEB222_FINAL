Explain the concept of Responsive Design. What does the term mean?

Responsive design is a strict belief that any design wether it be a webpage or program
should always adjust itself to the behavior of its user and/or environment. Some examples of this would
be screen size, different web browsers, multiple platform support and so on.

How does responsive design relate to web technologies?

Responsive design is extremly important to web technologies. Some applications come preformatted
for whatever software they are going to be run on. Unfortunately for us, most web applications and sites
do not follow the same. They are directly related since web design is very tedious, even the smallest of pixels
matter. You could create a design that works flawlessly on windows but then open it up on your phone
and you realize its unreadable. We must always think ahead before the design to ensure the product will
be able to run and display properly on all content pages.

What did you learn about building web apps for different devices through your research?

I learned that we need to constantly adapt to different screen sizes and platforms. Sizes of screens can vary
from 800+ (desktops) to even 70ish (small phones). It is much easier to build web apps for desktops since we
generally code on a desktop. Once its time to take it over to a smaller device we may encounter problems
since it has not been built to support it.
